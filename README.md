# 🎯 CS Operations Management System

## 🚀 رفع المشروع على GitHub + Vercel (خطوة بخطوة)

---

### الخطوة 1 — إنشاء حساب GitHub
1. افتح [github.com](https://github.com)
2. اضغط **Sign up** وأنشئ حساباً مجانياً

---

### الخطوة 2 — رفع المشروع
1. بعد تسجيل الدخول، اضغط **+** ثم **New repository**
2. اسم المشروع: `cs-ops` (أو أي اسم تريده)
3. اختر **Private** إذا تريده خاصاً
4. اضغط **Create repository**
5. اضغط **uploading an existing file**
6. اسحب جميع الملفات من هذا المجلد وارفعها
7. اضغط **Commit changes**

---

### الخطوة 3 — النشر على Vercel
1. افتح [vercel.com](https://vercel.com)
2. اضغط **Sign up** وسجّل بحساب GitHub مباشرة
3. اضغط **Add New Project**
4. اختر الـ Repository اللي أنشأته `cs-ops`
5. اضغط **Deploy** — خلال دقيقة يعطيك رابطاً جاهزاً ✅

---

### 🔄 كيف تحدّث الكود في كل مرة؟

1. افتح مشروعك على GitHub
2. اضغط على ملف `src/App.jsx`
3. اضغط أيقونة ✏️ (Edit)
4. احذف كل المحتوى والصق الكود الجديد
5. اضغط **Commit changes**
6. Vercel يحدّث الرابط تلقائياً خلال **30 ثانية** ✅
